Why iframes?

No screenshots — this is the actual responsive output, the same code that ships, just rendered into a 375 px viewport.

Why full height?

Lets you see the full page rhythm — hero → trust strip → features → REST API → stats → about — without scrolling inside a tiny window.

Auto-sizing

JS reads the iframe's contentDocument.scrollHeight after load + whenever the layout reflows, then sets the iframe height to match. ResizeObserver catches font reflow and accordion changes.
