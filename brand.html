Total Uptime — Brand system v1

Keep the Total Uptime name. Inherit the Edgenexus visual energy.

This is the visual system we're proposing for the merged brand — built from the Edgenexus palette and Montserrat type, applied to the existing Total Uptime site architecture (using the same Tailwind module the dev team already runs). Two sample pages prove it works end-to-end; this document explains the rules so it stays consistent as new pages and campaigns are built.

In scope

Colour, type, components, mobile menu fix

Built on

Tailwind CSS v3, Alpine.js — same stack as totaluptime-next

Out of scope

Logo refresh, illustration system, mascot — phase 2

Copy

Verbatim Total Uptime where authoritative; placeholders elsewhere — copywriter to finalise

How to use this doc: the palette, type and component sections describe the system. The accessibility section is the rule book — it's the result of running a WCAG 2.1 contrast pass over every brand pairing the design uses. The roadmap section flags work the client called out for phase 2.

Foundations · 03

Accessibility

Colour contrast audit — WCAG 2.1.

Every brand pairing the design actually uses, scored against WCAG 2.1's three text thresholds. The numbers below come from scripts/contrast.js — a single source of truth that we re-run whenever a token changes.

AAA · ≥ 7 : 1

Body copy, paragraphs, long-form. Highest bar — use for anything someone reads for more than a few seconds.

AA · ≥ 4.5 : 1

UI text, labels, button copy under 18 pt. Industry-default minimum.

AA Large · ≥ 3 : 1

Display headlines (≥ 24 pt, or ≥ 19 pt bold). Decorative type only.

Avoid — fails WCAG

These combinations exist in the palette but must not be used for text. The system enforces this in components (e.g. .eyebrow defaults to brand-blue, not brand-cyan).

Outstanding decision: the cyan hero surface (surface-cyan) inherently fails WCAG for body text. We've kept it for v1 because the hero only uses display-size copy with text-shadow, but the long-term call is either to (a) deepen the cyan surface to #0192C7, or (b) move heroes to surface-blue and use cyan as an accent only. Edgenexus mostly does the latter — happy to follow.

Voice & tone

Words — owned by the copywriter.

Copy is intentionally out of scope for this brand pass. Where the prototype already carries authoritative Total Uptime copy (hero, section eyebrows, section H2s, About paragraph), it's there verbatim. Where placeholder copy was needed to keep layouts honest, it's flagged. The copywriter has a clean canvas.

Reference 01 — current Total Uptime

"Control any application anywhere!"

Confident, direct, slightly playful punctuation. The current site's strongest line.

Reference 02 — Edgenexus tone

"Simplifying complex application delivery."

Calmer, more enterprise. Leans on jargon-free clarity.

Hypothesis for the merged voice

Technically credible, conversationally human.

As leadership called out: "permission to be more creative and expressive within what is traditionally a very safe space." Final tone of voice doc to be authored separately.

What's next

Phase 2 — what we're not doing yet, and why.

Leadership called out four expressive systems for phase 2. Locking the palette + type first is the right order: every phase-2 asset will be built on the same tokens you see above, which means nothing has to be reworked when illustration and mascot land.

  1. 1

    Logo refresh

    Total Uptime wordmark aligned to the new visual system.

    Re-cut on the new brand-blue token, with optional mark / monogram exploration. Until this lands, the current wordmark stays in place across the site.

  2. 2

    Line-art illustration evolution

    Evolve Edgenexus' line-art system to cover Total Uptime's product range.

    Diagram-friendly, scalable to product pages, decks and one-pagers. Uses the same coral / cyan / mint accent logic the palette defines.

  3. 3

    Custom stick-figure system

    A unique, ownable illustration treatment for human moments.

    For onboarding flows, empty states, support, careers, brand campaigns. Differentiates from the line-art system which stays diagrammatic.

  4. 4

    Mascot character

    Long-term storytelling and campaign identity.

    Worth its own brief — character, personality, range of poses, social presence. The palette and type here become its dress code.
