Total Uptime — navigation states

Every menu / navigation state rendered statically and visible at once. Use this URL for html.to.design instead of the homepage — that way the plugin captures the mega menu, mobile drawer, expanded accordion, the floating “On this page” pill and the bottom sheet, all in a single import.

Tip: resize your browser to at least 1500 px wide (so the 1440 px desktop frames render fully) before clicking the html.to.design extension. Desktop frames are exactly 1440 px wide, matching the standard Figma artboard. Mobile frames are 375 px.
